BoxBot4k
========

Description
-----------

The goal of the game to push boxes around a maze and try to put them in designated locations (squares with red pattern) 
by controlling a robot. 
Only one box may be pushed at a time, and boxes cannot be pulled. 
The game contains 50 levels.

Instructions
------------

Game controls
- Arrow keys: move the robot
- Backspace: restart level
- PgUp/PgDn: go to next/previous level
